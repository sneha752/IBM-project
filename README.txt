
# 🏥 Health AI Dashboard

A personal health management web app built with **Flask**, **Tailwind CSS**, and **Machine Learning** to help users:
- 📊 Analyze health reports
- ⌚ Integrate smartwatch data
- 🧠 Get AI-powered health risk predictions
- 👩‍⚕️ Find and contact doctors

---

## 🚀 Features
- 🔒 Secure local authentication with bcrypt
- 📄 Health report upload & automatic analysis
- 📈 AI-powered health risk prediction (Decision Tree Classifier)
- ⌚ Smartwatch data integration (CSV upload)
- 🤖 Built-in chatbot for doctor search
